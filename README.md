# 🌦️ Weather Zen

> **A clean, responsive, and user-friendly weather application built using HTML, CSS, and JavaScript.**

---

# 📖 Project Overview

Weather Zen is a simple weather forecasting web application developed as a **Computer Fundamentals Semester Project**.

It allows users to search for any city and view its weather information through an attractive and responsive interface.

The project demonstrates the use of:

- HTML page structure
- CSS styling
- JavaScript DOM manipulation
- API Integration
- Responsive Web Design

---

# ✨ Features

✅ Search weather by city

✅ Real-time weather information

✅ Temperature display

✅ Weather description

✅ Humidity

✅ Wind Speed

✅ Weather Icons

✅ Responsive UI

✅ Clean and Modern Design

---
